# Culebra E-Bike Map

An interactive map for e-bike guests exploring the island of Culebra, Puerto Rico.

## About

This project provides a digital map designed for e-bike rental guests, helping them navigate Culebra's roads, beaches, and points of interest. Whether you're heading to Flamenco Beach or exploring the wildlife refuge, this map has you covered.

## Features

- Interactive map of Culebra island
- E-bike friendly routes and trails
- Points of interest (beaches, restaurants, viewpoints)
- Estimated travel times by e-bike
- Safety tips and road conditions

## Getting Started

Visit the published map link provided with your e-bike rental to start exploring Culebra.
